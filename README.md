# CDOpoku_WEB101

A CRM-themed **Customer Journey Design Workshop** landing page built with HTML, CSS, and JavaScript.

This was my **first website built from scratch** as part of **CodePath Web Development 101**, where I learned the fundamentals of structuring pages, styling with CSS, and adding interactivity with JavaScript.

---

## Project Overview

This project is a single-page landing site for a fictional Customer Journey Design Workshop. It introduces the event, shares the schedule and journey mapping content, links to helpful CRM resources, and lets visitors RSVP through an interactive form.

The page combines layout, visual design, and JavaScript features such as form validation, a success modal, dark mode, and accessibility options.

### The Problem

Many students, entrepreneurs, and early professionals do not fully understand what CRM is, how it is used in the real business world, or how customer relationships connect to growth, marketing, sales, and long-term success.

### The Solution

This website presents a 3-day CRM event designed to guide students, entrepreneurs, and early professionals through the basics of CRM, customer journey mapping, and real-world business strategy. It also connects attendees to helpful resources from major companies and industry tools so they can keep learning after the event.

---

## Features

- **Navigation bar** — Fixed top navigation with smooth scrolling to About, Schedule, Journey Mapping, Resources, and RSVP sections
- **Header section** — CRM logo, event title, call-to-action buttons, and an embedded YouTube video
- **About the Event** — Brief introduction to the workshop theme
- **Problem & Solution** — Explains the CRM knowledge gap and how this 3-day event addresses it
- **Event Schedule** — Three agenda cards (CRM Briefing, Journey Mapping, Collaborative Workshop)
- **Journey Mapping** — Workshop content with CRM-themed images
- **Useful Links** — Curated external resources on CRM, design systems, and customer journey mapping
- **RSVP section** — Interactive form with validation, live participant list, and dynamic RSVP count
- **Success modal** — Personalized thank-you message after a valid RSVP, with optional logo animation
- **Dark mode toggle** — Switches the page between light and dark themes
- **Reduce Motion button** — Accessibility control to turn off modal animations
- **Fixed bottom action bar** — Quick links for **Learn More** and **Email Our Team**

## How to Run Locally

1. **Clone or download** this project to your computer.
2. **Open the project folder** (`CDOpoku_WEB101`).
3. **Launch the site** using one of these methods:
   - Double-click `index.html` to open it in your web browser, or
   - Use the **Live Server** extension in VS Code or Cursor for live reload while editing.

No installation or `npm install` step is needed.
