# CDOpoku_WEB101

A CRM-themed Customer Journey Design Workshop website built with HTML, CSS, and JavaScript.

## Problem

Many students, entrepreneurs, and early professionals do not fully understand what CRM is, how it is used in the real world, or how customer relationships connect to growth, marketing, sales, and long-term success.

## Solution

This website presents a 3-day CRM event that introduces students, entrepreneurs, and early professionals to CRM, customer journey mapping, and real-world business strategy. It also connects attendees to helpful CRM resources from major companies and industry tools.

## Features

- Navigation bar with smooth section links
- CRM event landing page layout
- Embedded CRM tutorial video
- Schedule, journey mapping, resources, and RSVP sections
- RSVP form with success modal
- Dark mode toggle
- Reduce motion option
- Fixed bottom action bar

## Technologies Used

- HTML
- CSS
- JavaScript
